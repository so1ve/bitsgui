# BITSGUI

简体中文 | [English](./README_EN.md)

一个简单的 BIT 校园网（10.0.0.55）登录/登出桌面应用。使用 Tauri、Vue 3 和 TypeScript 构建。

## 下载

### 预编译二进制文件

从 [GitHub Releases](../../releases) 下载最新版本。

### 从源码编译

请参考 [Tauri 官方文档](https://tauri.app/)。
